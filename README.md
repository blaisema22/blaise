# Manishimwe Blaise — Personal Portfolio

A modern, responsive personal portfolio website built with React + Vite.

## 🚀 Quick Start

```bash
# 1. Install dependencies
npm install

# 2. Start dev server
npm run dev

# 3. Open in browser
# http://localhost:5173
```

## 📦 Build for Production

```bash
npm run build
```

## ☁️ Deploy to Vercel

1. Push this folder to a GitHub repository
2. Go to https://vercel.com → New Project
3. Import your GitHub repo
4. Click **Deploy** — done!

## 📁 File Structure

```
blaise-portfolio/
├── index.html          ← HTML entry point
├── package.json        ← Dependencies & scripts
├── vite.config.js      ← Vite config
└── src/
    ├── main.jsx        ← React entry point
    └── App.jsx         ← Full website component ⭐
```

## ✏️ Customization

All content is in `src/App.jsx`. Edit the data arrays at the top:
- `SKILLS` — skill names and levels
- `PROJECTS` — your project list
- `EDUCATION` — school history
- `ICDL_MODULES` — certificate modules

## 🛠 Tech Stack

- React 18
- Vite 5
- Pure CSS-in-JS (no extra libraries needed)
- Google Fonts (Playfair Display + Inter)
