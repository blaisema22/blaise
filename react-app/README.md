# Blaise React scaffold

This folder is a minimal React (Vite) scaffold converting your static site and PHP views into React components.

Quick start:

1. From `react-app` run:

```bash
npm install
npm run dev
```

2. The scaffold provides components for Login, Signup, Parents, Trades and Trainees under `src/components`.
